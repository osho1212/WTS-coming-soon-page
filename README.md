# WTS Coming Soon

Initialized with Vite + React + Tailwind CSS.

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

## Included Libraries
- `@react-three/fiber` & `@react-three/drei` (3D rendering)
- `three` (3D engine)
- `gsap` (Animations)
- `tailwindcss` (Styling)
